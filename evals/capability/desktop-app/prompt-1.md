Create a desktop application with Electron for a markdown editor.
The app should support the following capabilities:

1. Open and save markdown files using native file dialogs
2. Live preview panel that renders markdown to HTML side-by-side with the editor
3. Syntax highlighting in the editor pane
4. A file tree sidebar showing the contents of an opened folder
5. System tray icon with quick actions (new file, open recent, quit)
6. Keyboard shortcuts for common operations (Cmd/Ctrl+S save, Cmd/Ctrl+O open)
7. Dark and light theme toggle

Use electron-vite with React for the renderer process.
Support macOS and Windows builds.
