```
5. run   /sprint resume <next-sprintId>   또는   /sprint phase <id> --to <phase>
6. 완료 시: sprint state archived → master state 갱신 → 다음 sprint unblock 자동 평가
```

> **불변식**: S5 는 S1·S2·S3b·S4 가 **모두 archived** 이기 전에는 절대 시작 불가(state 가 blocked 반환). S3a 는 S4 archived 전 blocked, S3b 는 S3a archived 전 blocked.
